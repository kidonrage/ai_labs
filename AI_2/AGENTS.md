# Commit Message Rules

- Формат: `[<префикс>] <текст>`
- Префикс возьми из последнего коммита в репозитории
- Язык: русский
- Залог: использовать форму "добавил/исправил/обновил"
- Сообщение после префикса начинать с заглавной буквы
- Длина сообщения: не более 40 символов (включая префикс)

Примеры:

- `[10] Добавил выбор стратегии чата`
- `[10] Исправил обработку ошибок`
- `[10] Обновил стили модального окна`
